    Import Images

    Portainer allows you to import the images that you have created in another Portainer instance or in the CLI of Docker and Docker Swarm.

    Importing Images

    To import an image using Portainer, go to Images and then click Import.

    Select your image, select the node where you will save the image and click Upload.
