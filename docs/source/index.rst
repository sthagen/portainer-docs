Portainer documentation
=======================

`Portainer`_ is a simple management solution for Docker.

For formal support of Portainer, please purchase a support subscription from `our website <https://www.portainer.io/products-services/portainer-business-support/>`_.

It consists of a web UI that allows you to easily manage your Docker containers, images, networks and volumes.
